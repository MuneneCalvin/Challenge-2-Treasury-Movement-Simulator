Treasury Movement Simulator

Multi-Currency Account Management & Transfer System

Project Scope

Built a comprehensive treasury management interface for handling multi-currency account operations across 10 virtual accounts spanning KES, USD, and NGN currencies. The system enables treasury teams to efficiently manage fund movements, track balances, and execute transfers with real-time foreign exchange conversions. The solution provides a production-ready interface with advanced features including live exchange rate calculations, transaction logging, and future-dated transfer scheduling.

Features Implemented

Account Management

  • 10 pre-configured accounts across 3 currencies
  • Account types: M-Pesa, Bank, Wallet
  • Real-time balance tracking
  • Currency-specific visual indicators
  • Account summary dashboard

Transfer Operations

  • Intuitive transfer modal interface
  • Live exchange rate conversion display
  • Insufficient balance validation
  • Optional transfer notes
  • Future-dated transfer scheduling

Transaction Management

  • Comprehensive transaction logging
  • Multi-filter system (currency, account, status)
  • Transaction status tracking

Key Assumptions Made

Exchange Rates

Static FX rates used: USD/KES (129.50), USD/NGN (460.25), with inverse calculations for reverse conversions.

Account Structure

Pre-defined account naming convention (Type_Currency_Number) with realistic balance distributions across account types.

Transaction Processing

Immediate processing for regular transfers, UI-only simulation for future-dated transfers with status tracking.
